# 📚 LibraryProject

A Django-based web application for managing a simple digital library. Features include book listing, search functionality, and form handling.

---

## 🚀 Features

- Semantic HTML with Django templating
- Secure headers and HTTPS configuration
- Book search using Django ORM
- Form submission example (`ExampleForm`)
- Fully modular app structure
